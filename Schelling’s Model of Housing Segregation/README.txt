Schelling Model of Housing Segregation

schelling.py: skeleton file.  Your code goes in this file.
utility.py: utility functions for dealing with grids.
README.txt: this file

This code implements a variant of Schelling’s model. Considering the next important features:
- Shape and composition of a city,
- Boundaries of a home’s neighborhood,
- Color of homeowners and criteria for similarity,
- Definition for satisfaction,
- Rule for relocating homeowners,
- Composition of a step in the simulation, and
- Stopping conditions for the simulation.
